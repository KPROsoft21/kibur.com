My Timeline

  1. Introduction HTML

    NOV 2018

    I first heard about HTML at a student lecture at my school, a senior at our school had a presentation about web development, and taught us about HTML and an editor for it called Adobe Dreamweaver, this laid the basic foundation in my programming career.

  2. Introduction to Python

    OCT 2019

    In 2019 I got my first Raspberry PI computer, a tiny desktop computer with every tool you need to power any software or hardware projects. With the computer came a printed tutorial on the basics of python and there started my journey into being a pro python coder.

  3. CSS & Advanced HTML

    MAR 2020

    COVID had shut down our school for the first time giving us "2 weeks" of unstructured break, me and my friend challenged eachother to advance our skills in a different programming language than python, the break which was supposed to end in 2 weeks turned out to be 8 months, after the COVID break I had become an HTML expert.

  4. JS, Flask & Django

    JUL 2020

    By July, I had realised that coding terminal apps is useless and my focus shifted from little computer based apps to scalable web apps. I discovered that my main programming language Python could be intgrated with frameworks to create full-blown websites. I aquired this skill in about 2 months creating a very successful (at the time) social media app called Table Chat; social media app that helped my classmates connect through the time of COVID. I also made my school a super advanced e-learning site for during COVID but I was brutally rejected by the assistant director saying "We don't give money back to students".

  5. PyGame & Pyglet

    NOV 2020

    By this time we had started to return to class part-time, I had noticed that the kids computer lab had been renewed. The kids computer lab was basically a place where you could play retro games with retro desktops. The Rennovation meant all the games had been deleted, so me and my friend decided we should make the games for them. I had just discovered a python module called Pygame to make 2D games and Pyglet to make 3D ones, I had recently aquired a new skill too, I reverse-engineered other people's code then the analysis I used taught me how to make my own games. After a while me and my friend had created a whole arcade of retro games for the kid's computer lab, in which some of them we sold to other kid's intersted in using these games on their own computers.

  6. Advanced Python Programming

    DEC 2020

    After my recent success with Table Chat & the Arcade games sale, I was inspired to make an impact in society with my programming skills and make some money along the way. I started noticing simple data management problems and worked on solving it. My first target was digitalizing the card catalog system for my school, after another rejection from the school to apply my system for an insignificant amount of money. Then I established my first money making start-up KPRO soft, I had taught myself a higher coding skill and database management, and taught my friends after. I sold code to many small businesses including my friend's parents super-markets or big inventory companies. Most business was conducted through contacts but we did help empower these businesses with reliable and affordable software.

  7. UNITY programming

    APR 2021

    After making everything I could imagine with pygame I felt like I needed a change. I consulted a few people about the best programming languages for game creation, the best choices were Unity or C. Unreal engine was too expensive to invest my precious time in high school. So I went to the second best option Unity, this is the only language I never really got a hand on, I used my reverse engineering techniques to recreate games and just change the dynamics of it. By stringing together pieces of code I was able to make a couple of really advanced games. This being a very complicated language, I sought help from a couple of friends I made on Github, guys with high paying jobs and degrees from top universities, these guys completed and corrected my code and in return I help them with their daily tasks at work from a child's point of view.

  10. Crypto Mining

    JAN 2022

    After christmass I stumbled on a youtube video about mining crypto.
